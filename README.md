# privateSanctum

Portafolio personal desarrollado con Vue 3 y Vite.

Descripción
-----------
Colección personal de proyectos y páginas de presentación, construida con Vue 3, TypeScript y Vite. Este repositorio sirve como plantilla y sitio de portafolio.

Demo
----
Sitio en vivo: https://erickvargas.dev

En desarrollo (repositorio): Abre un issue o PR si quieres ayudar con la implementación o despliegue.

Tecnologías
----------
- Vue 3
- Vite
- TypeScript
- HTML, CSS

Requisitos previos
------------------
- Node.js >= 16
- npm o yarn

Instalación
----------
```bash
npm install
```

Comandos útiles
---------------
```bash
# Ejecutar en desarrollo (hot-reload)
npm run dev

# Compilar para producción
npm run build

# Revisar tipos (si está configurado)
npm run type-check
```

Estructura relevante
--------------------
- `index.html` — punto de entrada HTML.
- `src/main.ts` — arranque de la aplicación.
- `src/App.vue` — componente raíz.
- `src/components/` — componentes reutilizables.
- `src/views/` — vistas/ páginas principales.
- `vite.config.ts` — configuración de Vite.

Configuración recomendada
-------------------------
- Editor: Visual Studio Code + Volar (desactivar Vetur).
- Extensión: Vue.js devtools en el navegador.

Cómo contribuir
---------------
1. Haz fork y crea una branch.
2. Abre un PR describiendo los cambios.
3. Mantén commits pequeños y claros.
